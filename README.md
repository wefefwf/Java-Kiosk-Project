## 작업 내용 
* Main을 기준으로 메뉴 담당 Order, 주문 내역 관리 담당 Order, 결제 담당 Payment 세가지 클래스로 나누어 
Main에서는 전체적인 흐름만 제어하고 기능은 각 클래스에서 담당한다.

* Main에 While문을 넣고 각 클래스의 메소드 별로도 While문을 넣어 Scanner에 잘못 입력될 시 
단계를 넘어가지 않도록 구현했다.

* 주문이 끝나고 영수증이 나오면 nextline()을 통해 enter를 눌러 다시 주문 처음으로 돌아갈 수 있다.
